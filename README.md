# ApeSwap-Golden-Banana

## Install dependency

```shell script
npm install
```

## Run unittest

Start local rpc:
```shell script
npm run testrpc
```

Start to run unittests:
```shell script
npm run test
```

## Flatten

```shell script
npm run flatten
```

## Configure
Create a .env file with the following values

```
BSCSCAN_KEY=[YOUR KEY]
DEPLOYER_KEY=[YOUR KEY]
```
